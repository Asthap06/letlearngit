## Basics of java

1. ADDITION
```java
 class sum{

    public static void main(String[] args){
        int a=5;
        int b=4;

        System.out.println("the sum is " + (a+b));
    }
}
```
2. SUBSTRACTION
```JAVA
public class diff{

    public static void main(String[] args){
        int a=5;
        int b=3;

        System.out.println(a-b);
    }
}
```
